# AI Output Review Checklist

Use this before sharing or applying AI-generated output.

- [ ] The output matches the provided facts.
- [ ] Assumptions are clearly marked.
- [ ] No secrets are included.
- [ ] Commands are safe to run.
- [ ] Destructive commands require approval.
- [ ] Documentation references are verified.
- [ ] Customer-facing tone is calm and clear.
- [ ] Final recommendation has been reviewed by a human.
