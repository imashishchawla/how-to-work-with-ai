## About

This repository is the companion to the **AI for Engineers** sessions delivered by Ashish Chawla, covering how infrastructure and platform engineers can adopt AI tools safely and effectively. Each session is self-contained and builds on the previous one.

Topics covered across sessions include: AI basics, key terms, local vs. online LLMs, model selection, coding agents (Claude Code, Codex, OpenCode, IBM Bob), prompt engineering, infrastructure-as-code with AI, and security and governance.
